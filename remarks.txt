КОГДА НЕЛЬЗЯ ПРЕРЕНДЕРИТЬ ДЛЯ ТОГО:
чтобы страница была предрендеримой, любые два пользователя, напрямую обращающиеся к ней, должны получить с сервера один и тот же контент.
export const prerender = false;

КОГДА НЕЛЬЗЯ ОТОБРАЖАТЬ НА САТРНИЦЕ:
Это полезно, если ваша страница не может быть отображена на сервере 
(поскольку вы, например, используете глобальные переменные только для браузера document)
export const ssr = false;

КОГДА НЕ НАДО ОТОБРАЖАТЬ В ИНТЕРАКСТИВНУЮ СТРАНИЦУ
Обычно SvelteKit преобразует HTML-код, отображаемый на сервере, в интерактивную страницу, отображаемую на стороне клиента (CSR).
Некоторые страницы вообще не требуют JavaScript
export const csr = false;

УДАЛЕНИЕ КОНЕЧНЫХ КОСЫХ ЧЕРТ
export const trailingSlash = 'always';
